# AI Search Results Blocker

This blocklist removes websites that contain AI-generated images or AI-generated content from search engine results. When used with **uBlacklist**, results from these domains will be hidden on search engines such as Google, DuckDuckGo, and Bing, helping keep image searches cleaner and focused on non-AI content.

## Manual Import

1. Open the **uBlacklist options** page.
2. Enable desired search engines via **Other search engines / SERPINFO** if needed.
3. Scroll to **Subscription**.
4. Click **Add a subscription**.
5. Enter a name (e.g. `Main AI blocklist`).
6. Paste this URL into the **URL** field:

```
https://raw.githubusercontent.com/Bokan96/uBlockOrigin-HUGE-AI-Blocklist/refs/heads/main/list_uBlacklist.txt
```

7. Click **Add**.
8. Set update interval (e.g. 1 hour).

# iOS / iPadOS (Safari)

1. Install **uBlacklist for Safari**.
2. Enable the extension in **Settings → Safari → Extensions**.
3. Allow permissions for your search engine.
4. Open **Extension Settings**.
5. Scroll to **Subscription**.
6. Click **Add a subscription**.
7. Enter a name (e.g. `Main AI blocklist`).
8. Paste the URL:

```
https://raw.githubusercontent.com/Bokan96/uBlockOrigin-HUGE-AI-Blocklist/refs/heads/main/list_uBlacklist.txt
```

9. Click **Add** and set update interval.
